# Тренажер устного французского

Локальный тренажер устной подготовки по `Edito A2`, сфокусированный на юнитах 7-12.

## Как открыть

Откройте файл `index.html` в браузере. Приложение не требует интернета и зависимостей; прогресс сохраняется в `localStorage` браузера.

## Как поделиться с группой

Проект можно выложить как бесплатный статический сайт. Инструкция: `DEPLOY.md`.

Публиковать нужно только:

- `index.html`
- `styles.css`
- `app.js`
- `README.md`
- `.gitignore`

PDF учебника не нужен для работы сайта и не должен публиковаться.

## Что уже есть

- Активные модули с 7-го по 12-й юнит.
- Устные вопросы по темам.
- Обязательные слова из страниц `Vocabulaire` для активных юнитов.
- Группы словаря сохранены как в фиолетовых блоках учебника.
- Готовые ответы трех уровней сложности.
- Таймер на 45 секунд для проговаривания.
- Конструктор ответа с подсветкой использованных слов.
- Режим скрытого ответа для тренировки устной памяти.

## Как будем развивать

Когда будут известны конкретные билеты, в `app.js` можно добавить вопросы и готовые устные ответы под каждый билет.
